# Large Model Learning

本仓库记录了我在学习自然语言处理（NLP）、Transformers、检索增强生成（RAG）、智能体（Agent）以及 FastAPI 等技术方面的过程和进展。

## 涉及的技术

- **自然语言处理（NLP）**：涉及文本预处理、分词、语言模型微调等相关技术和算法。
- **Transformers**：深入研究 Transformer 模型，包括 BERT、GPT、T5 等架构，及其在 NLP 任务中的应用。
- **检索增强生成（RAG）**：将检索方法与生成模型相结合，用于提高语言任务的表现。
- **智能体（Agent）**：探索强化学习和智能体架构，用智能体解决实际问题。
- **FastAPI**：学习如何构建快速高效的 API，将机器学习模型与后端服务进行集成。
